MB.Ch1.12. Write a function that calculates the proportion of values in a vector x that exceed some value cutoff.

prop_cutoff <- function(x,a){
  tol = 0
  for(i in 1:length(x)){
    if(x[i]>a){
      tol=tol+1
    }
  }
  tol/length(x)
}
